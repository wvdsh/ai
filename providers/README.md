# Provider Plugins

This directory contains provider-specific plugin packages for AI coding tools.

## Skills

Do not edit skill files under `providers/*/plugin/skills/` manually. They are
generated by `scripts/sync.js` from the canonical Wavedash skill source served
by `https://docs.wavedash.com/.well-known/skills/index.json`.

Provider folders should only contain packaging files that are specific to that
agent platform.
